## **Decoding Wolfram**

This function converts a Wolfram string to a readable string.

### *Prototype*

```javascript
const { text, lenOffset } = encodeWolfram(wolframString);
``` 

### *Parameters*

* **wolframString** - The string written in Wolfram syntax to be converted

### *Return*

A Javascript object as:
* **text** - The converted string  
* **lenOffset** - A number indicating how many characters the cursor has been moved with respect

### *Example*

```javascript
const WolframString = "{{A, A \\[Implies] B}, {B}}";
const { text, lenOffset } = decodeWolfram(WolframString);
console.log("Converted text: " + text);
console.log("New cursor position with respect to the original cursor: " + lenOffset);
```

## **Encoding Wolfram**

This function converts a string to Wolfram string that can be understood by the Wolfram API.

### *Prototype*

```javascript
const text = encodeWolfram(wolframString);
``` 

### *Parameters*

* **wolframString** - The string to be converted

### *Return*

A Javascript object as:
* **text** - The converted string in Wolfram syntax 

### *Example*

```javascript
const WolframString = "{{A, A ⇒ B}, {B}}";
const text = encodeWolfram(WolframString);
console.log("Converted text: " + text);
```

## **Format Problem as JSON**

This function return the problem text in JSON format from the problem object.

### *Prototype*

```javascript
const text = formatProblemAsJSON(problem);
``` 

### *Parameters*

* **problem** - The problem object to be formatted.

### *Return*

* **text** - The resulted problem text in JSON format.

### *Example*

```javascript
const problem = {
    Hypothesis: "{A, A ⇒ B}",
    Goal: "{B}"
};
const text = formatProblemAsJSON(problem);
console.log(text);
```
