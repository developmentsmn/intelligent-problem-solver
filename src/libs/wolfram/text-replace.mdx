# Wolfram Text Converter
---


## **Decoding Wolfram**

This function converts a Wolfram string to a readable string.

### *Parameters*

* **text** - The string written in Wolfram syntax to be converted

### *Return*

A Javascript object as:
* **text** - The converted string  
* **lenOffset** - A number indicating how many characters the cursor has been moved with respect

### *Example*

```javascript
const WolframString = "{{A, A \\[Implies] B}, {B}}";
const { text, lenOffset } = decodeWolfram(WolframString);
console.log("Converted text: " + text);
console.log("New cursor position with respect to the original cursor: " + lenOffset);
```

## **Encoding Wolfram**

This function converts a string to Wolfram string that can be understood by the Wolfram API.

### *Parameters*

* **text** - The string to be converted

### *Return*

A Javascript object as:
* **text** - The converted string in Wolfram syntax 

### *Example*

```javascript
const WolframString = "{{A, A ⇒ B}, {B}}";
const text = encodeWolfram(WolframString);
console.log("Converted text: " + text);
```
